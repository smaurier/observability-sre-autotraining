Les 3 Piliers de l'Observabilite

Cliquez sur un pilier pour explorer ses details

📋
Logs
Evenements textuels horodates decrivant ce qui s'est passe dans le systeme

Ce que ca capture

Evenements discrets, erreurs, informations de debug, requetes HTTP, changements d'etat

Outils

  • Elasticsearch / OpenSearch
  • Loki (Grafana)
  • Fluentd / Fluent Bit
  • Datadog Logs

Exemple

2026-03-09T10:23:45Z [ERROR] OrderService: Payment failed for order #4821 - timeout after 30s

📊
Metriques
Mesures numeriques aggregees dans le temps pour surveiller les tendances

Ce que ca capture

Valeurs numeriques: taux de requetes, latence, utilisation CPU/memoire, taux d'erreur

Outils

  • Prometheus
  • Grafana
  • InfluxDB
  • Datadog Metrics

Exemple

http_requests_total{method="POST", status="500"} 142
http_request_duration_seconds{quantile="0.99"} 2.3

🔗
Traces
Suivi du parcours d'une requete a travers les services distribues

Ce que ca capture

Parcours complet des requetes, latence par service, dependances, goulots d'etranglement

Outils

  • Jaeger
  • Zipkin
  • Tempo (Grafana)
  • OpenTelemetry

Exemple

TraceID: 3a2f7c...
├─ API Gateway (12ms)
│ ├─ Auth Service (3ms)
│ └─ Order Service (45ms)
│ └─ Payment Service (120ms)

Logs — Evenements textuels
Metriques — Mesures numeriques
Traces — Parcours distribue
